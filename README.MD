# Versão em Português

## O que o script faz?

Esse script lê todos os arquivos PNG de uma pasta. Nesse processo, redimensiona cada uma das imagens para a resolução correta.
Para começar, uma janela irá perguntar qual é a resolução atual das suas imagens (que podem incluir icones ou qualquer imagem PNG).
Posteriormente, será gerado para cada imagem suas versões em resoluções menores.

### Por exemplo:

```
Em uma pasta, você tem todo o seu material na resolução XXHDPI. 
O script lerá todo material e converterá cada imagem para XHDPI, HDPI e MDPI.
```

## Como utlizar?

1. Insira o script na pasta Adobe Photoshop > Presets > Scripts (que pode variar de acordo com sua versão);
    a) Você não precisa fazer isso mais de uma vez.
2. Abra o Photoshop e navegue até Arquivo > Scripts > Android Icons;
3. Selecione a resolução atual do seu material;
4. Selecione a pasta que contenha o seu material;
5. Pronto! Dentro da pasta do seu material, estarão salvos as resoluções menores em suas respectivas pastas.

## Como alterar esse script?

A qualquer momento você pode alterar esse script conforme suas necessidades. 
Porém, ao modificar e reenviar esse script para o GitHub, coloque aqui as mudanças.

# English Version

## What does this script?

This script reads all PNG files inside a folder. In this process, it resizes each image to the right resolution.
To begin, a window will ask you what is the current resolution of your images (which may include your icons or any PNG image).
Subsequently, smaller dimensions versions will be generated for each image.

### For example:

```
Insides a folder, you have all of your material in XXHDPI resolution.
The script will read all material and will convert each image to XHDPI, HDPI e MDPI.
```

## How to use?

1. Install the script copying the **Android Icons.jsx** file to Adobe Photoshop > Presets > Scripts folder (which may vary by version);
    a) You don't have to do this more than once.
2. Open the Photoshop and navigate to File > Scripts > Android Icons;
3. Select the current resolution of your materials;
4. Select the folder that contains your materials;
5. Ready! Inside your materials folder, they will be saved as smaller resolutions in their respective dimensions.

## How to change this script?

At any time you can change this script according to your needs.
However, when modifying and resending this script to GitHub, put the changes here.
